# Third-party content

Wallscape can search and download media from Wallhaven and MotionBGS when requested by the user. Neither service is operated by or affiliated with Wallscape, and no third-party wallpaper is bundled with the application.

Copyright and licensing remain with each wallpaper's creator or rights holder. Availability through a website or API does not automatically grant permission to redistribute, modify, or use content commercially. Review the source page and the service's current terms before using or sharing downloaded media.

Wallscape stores downloaded files locally and provides links back to source pages where available. Report integration failures to Wallscape; report content ownership or service-policy concerns to the relevant service.
