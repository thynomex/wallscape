# Support

Use GitHub Issues for reproducible bugs and feature requests. Please include the Wallscape version, Windows version, GPU, monitor layout, steps to reproduce, expected result, and actual result.

Logs may contain local file paths or wallpaper names. Review and redact them before posting. Do not publish credentials or private media.

General Windows, Tauri, mpv, Wallhaven, and MotionBGS support is outside the project's scope. Security concerns belong in the private process described in [SECURITY.md](SECURITY.md).
